# Jonah Lage - Portfolio Website

## Overview

A modern, secure, and feature-rich portfolio website showcasing software development projects and skills. Built with vanilla HTML, CSS, and JavaScript for optimal performance and security.

## Features

### Design
- **Dark Theme**: Sophisticated color palette with purples, grays, blacks, and dark blues
- **Custom Cursor**: Interactive cursor with smooth following animation
- **Gradient Orbs**: Animated floating background elements with parallax effects
- **Glitch Effect**: Eye-catching text animation on hero title
- **Smooth Animations**: Intersection Observer API for scroll-triggered animations
- **Parallax Scrolling**: Dynamic background movement on scroll
- **Hover Effects**: Interactive cards and buttons with scale and glow effects
- **Responsive Design**: Mobile-first approach with optimized layouts for all devices

### Security
- **Content Security Policy**: Prevents XSS attacks
- **X-Frame-Options**: Prevents clickjacking attacks
- **X-Content-Type-Options**: Prevents MIME-type sniffing
- **Referrer Policy**: Controls referrer information
- **Input Sanitization**: XSS prevention for future form inputs
- **Right-click Protection**: Optional image protection

### Performance
- **Vanilla JavaScript**: No framework overhead
- **Optimized CSS**: Hardware-accelerated animations
- **Lazy Loading**: Intersection Observer for efficient rendering
- **Performance Monitoring**: Built-in load time tracking

### Accessibility
- **Semantic HTML**: Proper document structure
- **Keyboard Navigation**: Full keyboard support
- **ARIA Labels**: Screen reader friendly (ready for expansion)
- **Focus Indicators**: Clear focus states for interactive elements

## Deployment

This site can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

### GitHub Pages Setup

1. Go to repository Settings
2. Navigate to Pages section
3. Select `main` branch as source
4. Save and wait for deployment
5. Access at `https://jonahlage.github.io/portfolio/`
